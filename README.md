## Installation:
1. Clone repo with: `git clone https://github.com/kbegiedza/dotfiles.git`
2. (optional) Install dev-tools, vscode, brave browser, etc: `./dotfiles/fresh_install.sh`
3. Install dotfiles: `./dotfiles/install.sh`
